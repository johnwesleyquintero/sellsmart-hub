# SellSmart Hub Design System

## Design Tokens

### Color Scheme
- Primary: #2D5BFF (Brand Blue from favicon.svg)
- Secondary: #00C9A7 (AI Accent from og-image.svg)
- Neutral: #1A1D26 (Dark Theme Base)
- Surface: #2D3250 (Component Background)
- Error: #FF4D4D
- Success: #00CC88

### Typography
- **Primary Font**: Inter (System UI stack)
- **Code Font**: Lato
- Base Size: 16px
- Heading Ratio: 1.333 (Perfect Fourth)

## Logo Usage
1. Always maintain 24px clearance space
2. Use monochrome version on colored backgrounds
3. Minimum size: 32px digital, 0.5" print

## UI Components
- Button Radius: 6px
- Card Elevation: 0 2px 8px rgba(0,0,0,0.1)
- Input Border: 1px solid #2D3250

## Documentation Site Requirements
1. Dark theme first implementation
2. Interactive component playground
3. Versioned design assets
4. Contrast ratio ≥ 4.5:1
5. Mobile-first responsive layout
