### CSV Data Mapper

**Status**: ‚úÖ Active
**Version**: 1.0.0

üîç **Description**: A reusable React component for mapping CSV headers to target metrics with validation.

**Features**:

- Generic TypeScript implementation for flexibility
- Dynamic form generation based on target metrics
- Comprehensive validation including:
  - Required field checks
  - Duplicate mapping detection
  - Input type validation
- Error handling with user feedback
- Responsive UI with shadcn/ui components

**Usage**:

```tsx
<CsvDataMapper
  csvHeaders={csvHeaders}
  targetMetrics={[
    { key: 'sales', label: 'Total Sales', required: true },
    { key: 'units', label: 'Units Sold', required: true },
    { key: 'refunds', label: 'Refunds', required: false },
  ]}
  onMappingComplete={handleMappingComplete}
  onCancel={handleCancel}
/>
```

**Dependencies**:

- React 18+
- TypeScript 5.0+
- shadcn/ui components
